Privacy Policy

Kina Influencer Agent helps organize creator campaigns and related campaign data. This policy explains the intended handling of information connected to the service.

Information Used

Kina may use creator profile details, campaign details, content metadata, and performance information when a creator or authorized user connects an account.

TikTok Data

TikTok data is used to support creator discovery, campaign deliverable matching, and campaign reporting. Kina should only request data needed for those purposes.

Reddit Data

Reddit data is used to support public community research, topic discovery, campaign fit review, draft preparation, and human-reviewed reporting. Kina should not collect private Reddit data, automate spam, sell raw Reddit data, or use Reddit data to train a model without approved permission.

Secrets and Tokens

Client secrets, access tokens, refresh tokens, and API credentials must be stored on a secure backend, not in public website code.

Data Removal

Users may request that connected account data be disconnected or removed by contacting the site owner or using account controls when available.
