# Kina Influencer Agent

Static landing page and TikTok API setup pages for Kina Influencer Agent.

## Pages

- `index.html` - public landing page
- `tiktok-api.html` - TikTok API setup and developer review support
- `privacy.html` - privacy policy
- `terms.html` - terms of service

## Local Preview

Open `index.html` in a browser. No install or build step is required.

## GitHub Pages

After pushing this folder to GitHub, enable Pages from the repository settings and publish from the default branch root.

## Security

Do not put TikTok client secrets, access tokens, refresh tokens, or token exchange logic in these frontend files. The OAuth start and callback routes should be implemented on a backend.
