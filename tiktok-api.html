OAuth Redirect

Configure this redirect URI in the TikTok Developer app after your production domain is connected.

https://your-domain.com/api/auth/tiktok/callback

The callback must be handled by your backend. Do not process TikTok tokens in the browser.

Connect TikTok

When the backend is ready, this button should start the OAuth authorization flow from your server.

Connect with TikTok

This static page does not include a client secret, access token, refresh token, or token exchange request.

Requested Data Use

Basic profile information Used to identify the connected creator account and show the account name inside Kina.
Video list and content metadata Used to match campaign deliverables to creator posts that the creator has authorized.
Performance insights Used to summarize campaign results for the brand or team managing the campaign.

Security Commitments

  • No TikTok client secret is stored in frontend code.
  • No access token or refresh token is written to this static site.
  • Users can disconnect their TikTok account from the backend integration when available.
  • Only creator-authorized data is used for campaign operations and reporting.
